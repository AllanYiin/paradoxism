### prompt以及chain_of_thought行為上的差異

```python
from paradoxism.base.agent import agent
from paradoxism.ops.base import *


@agent('gpt-4o',system_prompt='你是一位暢銷的童書作家')
def make_story_prompt(input_topic):
    """
    你的主要目標讀者會是5~9歲之間的兒童˙
    你需要以他們生活中可以理解的概念與行為來表哪你書中的內容
    避免過度艱澀的語言以及沒有明確因果關係的劇情

    Args:
        input_topic: (str) 輸入的故事主軸
    Returns:

    """
    results=prompt('請基於input_topic的概念設計出以擬人的動物角色所構成的繪本故事大綱',input_kwargs={'input_topic':input_topic})
    return results


@agent('gpt-4o', system_prompt='你是一位暢銷的童書作家')
def make_story_cot(input_topic):
    """
    你的主要目標讀者會是5~9歲之間的兒童˙
    你需要以他們生活中可以理解的概念與行為來表哪你書中的內容
    避免過度艱澀的語言以及沒有明確因果關係的劇情

    Args:
        input_topic: (str) 輸入的故事主軸
    Returns:

    """
    results = chain_of_thought('請基於input_topic的概念設計出以擬人的動物角色所構成的繪本故事大綱',
                     input_kwargs={'input_topic': input_topic})
    return results
```

**故事大綱：**

在一個陽光明媚的早晨，小兔子露露和小松鼠奇奇在森林裡玩耍時，發現了一張古老的地圖。地圖上標示著一個神秘的寶藏地點，這讓他們興奮不已。於是，他們決定展開一場尋寶冒險。

露露和奇奇一路上遇到了許多挑戰，比如要穿越一條湍急的小溪，還要爬上一座高高的山丘。在這些過程中，他們學會了互相幫助和合作，並且發現了彼此的優點。

當他們終於抵達地圖上的寶藏地點時，卻發現那裡並沒有金銀珠寶，只有一個小木箱。打開木箱後，他們看到裡面有一封信，是老貓頭鷹智者寫的。信中說明，真正的寶藏並不是金錢，而是他們在旅途中所學到的友誼、合作和勇氣。

### Concept Alignment
> - **比金錢更重要的事**：這可以指利於個人成長、情感、道德價值觀或其他在生活中被認為更有價值的事物，如家庭、友誼、真誠、愛、健康與幸福等。
> - **擬人的動物角色**：這些是具有人的品質和行為的動物角色，便於小朋友理解故事中的倫理和情感。
> - **繪本故事大綱**：涉及簡短的故事情節設計，適合5-9歲小孩，重點在於簡單易懂的語言和明確的因果關係。

### Chain-of-thought
> 1. 我需要設計一個簡單的故事，讓動物角色展示什麼東西比金錢更重要。
> 2. 我可以選擇一些常見且易於繪製的動物，例如兔子、小狗和貓。
> 3. 故事的主題應該集中在一個有趣且容易讓小孩理解的情節上，比如朋友之間的合作和支持。
> 4. 設定一個情境讓動物角色面臨金錢的誘惑與另一個更重要的選擇。
> 5. 結局應該清楚地傳達金錢並不是最重要的，友誼和關係等其他因素才是值得珍視的。

### 繪本故事大綱
故事名稱：朋友比金錢更閃亮

在一座小森林裡，住著三個好朋友：小兔子蘿蘿、小狗皮皮和小貓咪咪。他們最喜歡一起玩耍和探索森林的每個角落。一天，在森林深處，他們發現了一個閃亮的寶箱，裡面盛滿了金幣。

蘿蘿、皮皮和咪咪開始討論該怎麼處理這些金幣。蘿蘿認為，可以用這些金幣買很多胡蘿蔔，而皮皮則想要很多的玩具球。可是咪咪卻注意到了遠處的小河已經乾涸，小魚們正面臨困境。

咪咪提議，為什麼不拿金幣換取工具，好讓他們能夠幫助把河水引回來，讓小魚有個家呢？起初，蘿蘿和皮皮對這個提議有些猶豫，但想到他們平時在河邊和小魚們的快樂時光，他們決定一起努力。

最後，三位朋友利用金幣買來了鏟子和管道系統，成功地讓河水再次流淌，小魚們感恩地在河中游來游去。雖然他們沒有了金幣，但得以和更多朋友共渡快樂時光，讓他們明白到，朋友的陪伴和互助才是比金錢更重要的事。

### Rethink
>> 本次大綱確實有呈現“比金錢更重要的事”這個主題，小朋友能夠透過三位動物角色互助的行為，理解友誼、合作重要性。
>> 故事簡潔明了，適合5-9歲兒童的理解能力，且有明確的因果關係。
>> 儘管故事的背景簡單易懂，但故事情節的設計本可更多層次，例如涉及動物角色因為失去寶藏而感到短暫失落，但在幫助活化小河後得到的快樂更豐富。
>> 可以更具體描繪動物角色在選擇幫助小魚之前的小衝突，形成更有說服力的情感弧線。

### Chain-of-thought (Revised)
> 1. 保留三個動物角色與情節設定，讓小朋友可以輕易理解。
> 2. 增添動物角色對於失去用於個人享樂的金幣而感短暫失落的情景。
> 3. 更加描述他們團結起來後的成就感與快樂，形成情感上的反差。
> 4. 以一個結尾強調這次體驗帶給他們對於友誼和合作的永久價值。
